# Contributing to MindBridge

Thank you for your interest in MindBridge!

## How to Contribute
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/amazing-feature`).
3. Commit your changes (`git commit -m 'Add amazing feature'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request.

## Coding Conventions
- Use TypeScript for all extension logic.
- Follow PEP 8 for Python backend code.
- Ensure all new features are documented in the `docs/content/docs` folder.
